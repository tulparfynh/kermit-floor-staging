# Writing Templates

Use one shared topic plan and adapt per locale.

## Target Length

- EN: 900-1300 words
- TR: 900-1300 words

## Inline Media Density

- Decide image count autonomously from content depth:
  - 900-1050 words: 2-3 images
  - 1050-1300 words: 3-5 images
- Place visuals where they clarify a point, not as decoration.
- If user provides videos, place each playable video in the most relevant section.

## EN Template

1. Problem framing
2. Decision criteria and context
3. Practical guidance and implementation checks
4. Where SPC products fit (neutral and factual)
5. Practical wrap-up

Optional: add one contextual CTA sentence in section 4 or 5 when it helps the reader continue.

Suggested media placement:

- One early context image after section 1 or 2
- One practical/detail image inside section 3
- Optional comparison/detail image near section 4
- Optional playable user video near the section it supports

## TR Template

1. Problemin tanimi ve baglam
2. Degerlendirme kriterleri
3. Uygulamaya donuk pratik oneriler
4. SPC urunlerinin uygun oldugu noktalar (tarafsiz ve olgusal)
5. Pratik ozet ve kapanis

Opsiyonel: okuyucunun sonraki adimina gercekten yardimci olacaksa 4 veya 5. bolume tek bir baglamsal CTA cumlesi ekleyin.

Onerilen medya yerlesimi:

- 1 veya 2. bolumden sonra baglam gorseli
- 3. bolumde uygulama/detay gorseli
- 4. bolumde opsiyonel karsilastirma veya detay gorseli
- Kullanici videosu varsa ilgili bolume oynatilabilir video yerlesimi

## Locale Adaptation Rules

- Do not literal-translate EN to TR.
- Keep search intent aligned across locales.
- Keep examples and phrasing natural for each audience.
- Preserve shared topic meaning and conversion objective.
- Use proper Turkish characters in TR output (`ç, ğ, ı, İ, ö, ş, ü`).
- Use bold emphasis for high-signal terms or decision points only (for example: `**moisture risk**`, `**installation checklist**`).
- Keep emphasis sparse so highlighted text remains meaningful.

## Comparison Mode

Use comparison framing only when topic/keyword intent requires it.

- Comparison topics: alternatives, versus terms, evaluation queries.
- Non-comparison topics: explainers, how-to, checklist, maintenance guides.
